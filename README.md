## Timmy and mountains

# Setup and Installation



## Clone Repository
```
git clone git@github.com:bjvta/timmy-mountain.git && cd timmy-mountain
```


## Install Dependencies
### Add the virtualenv 

```
pip3 install virtualenv
virtualenv env
source env/bin/activate
```

### Install dependencies

```
pip3 install -r requirements.txt
```

## Run Tests
```
pytest
```
